Here are some functions you can use to command the robot.

pri.gripper_ctrl(command) - ctrl the gripper, param 'command' is string type.
pri.move(pos, quat) - set the target pose of the end effector/gripper, param 'pos' means the target position as input.
    Param 'quat' means the target quaternion as input.If you want to move the end effector to a target point, you need call this function.
    If you call this function, the gripper will move to this target.

Here are some properties you can use.
pri.can_pos - present the position of can object。
pri.can_quat - present the quaternion of can object。

Here are some rules you need note.
If there is a method not mentioned above, don't generate it.
If you want to grab an object, you need to let the gripper open before moving the arm.
